## Tips for Best Results

### For DALL-E 3 (ChatGPT Plus)
1. Copy the prompt exactly
2. Paste into ChatGPT
3. Wait for generation (30-60 seconds)
4. Download the image
5. If not satisfied, ask for variations: "Make it more colorful" or "Add more detail"

### For Leonardo.ai (Free)
1. Sign up at leonardo.ai
2. Choose "Leonardo Diffusion XL" model
3. Paste prompt
4. Set image dimensions to 16:9 (1280x720) for presentation
5. Generate (4 images per prompt)
6. Download your favorite

### For Midjourney (Discord)
1. Join Midjourney Discord
2. Use command: `/imagine prompt: [paste prompt here]`
3. Add `--ar 16:9` at the end for widescreen
4. Example: `/imagine prompt: cute anime girl waving --ar 16:9`

### For Bing Image Creator (Free)
1. Visit bing.com/create
2. Paste prompt
3. Click "Create"
4. Download generated images

## Troubleshooting

**Image doesn't match style?**
- Add "professional", "modern", "clean" to prompt
- Specify "3D illustration" or "flat design"

**Colors wrong?**
- Add "purple and blue gradient theme"
- Specify exact hex colors

**Too complex/busy?**
- Add "minimalist", "clean", "simple"
- Remove extra details from prompt

**Not tech-related enough?**
- Add "technology", "digital", "UI/UX design"
- Include "modern web application"
